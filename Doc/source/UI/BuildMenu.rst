BuildMenu
=========

Workflow
--------

``FBuildables`` is a ``DataTable`` that provide data to BuildMenu. When a button in the menu is clicked, ``Ability_Build``
is triggered, and the ``SoftObjectPtr`` static mesh is passed to create ``BinggyWorldBuildable``.
